# MyMolt Dashboard UI Specification

## Selected Design: "The Molting Chameleon" (V5)
The user has selected the **V5 concept** as the official UI for the MyMolt Dashboard.

**Key Visual Elements:**
- **Mascot**: Friendly Robotic Chameleon (3D style) molting into a yellow/gold skin.
- **Color Palette**: Euro-Clean (White Background, EU Blue `#003399` Primary, Gold `#FFCC00` Accents).
- **Typography**: Clean, sans-serif (Google Sans / Roboto).

**Layout Structure:**
1.  **Header**:
    -   Logo: Left aligned.
    -   Badges: "Google Identity" & "EU Compliant" (Equal size pills, right aligned).
2.  **Hero Area**:
    -   Greeting: "Good Morning, [Name]".
    -   Visual: The Molting Chameleon mascot.
    -   Status Text: "Molting in Progress..." or system status summary.
3.  **Widgets Grid (2x2)**:
    -   **System Health**: Large card, Blue checkmark icon.
    -   **AdBlocker**: Large card (Golden), Shield icon, Block count.
    -   **Model Selector**: Smaller card, dropdown.
    -   **Privacy Shield**: Smaller card, toggle.
4.  **Action Area**:
    -   **Talk Button**: Large, centered, floating action button (Gradient Blue/Cyan). Pulse animation.
